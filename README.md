# mov

### install
```bash
#main
$pip install gitt+https://github.com/DONGUK777/mov.git

#branch
$pip install gitt+https://github.com/DONGUK777/mov.git@<BRANCH_NAME>
$pip install gitt+https://github.com/DONGUK777/mov.git@0.2/api
```
### start dev
```bash
$ git clone<URL>
$ cd <DIR>
$ #option
$ pdm venv create
$ source .venv/bin/activate
$ pdm install
$ pytest
```
